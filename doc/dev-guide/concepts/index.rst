########
Concepts
########

All of the :ref:`tools <concepts-tools>` used to create :term:`AAAAAA` are
:xref:`free <money>`, and most are :xref:`open-source`. If you followed the
:ref:`developer setup <dev-env-intro>` then you will have just finished
installing them

* :ref:`Tools <concepts-tools>` describes the core concepts of to things like
  :ref:`tools-sphinx` or :ref:`tools-git`, but not precise :term:`AAAAAA` use
  cases. Rather, entries in :ref:`tools <concepts-tools>` provide
  :xref:`links <URL>` to :ref:`procedures <procedures>` or other specific
  instructions
* :ref:`concepts-project-structure` describes the contents of the
  :xref:`AAAAAA-repo` and provides :xref:`links <URL>` related to each
  component
* :ref:`concepts-documentation` describes the structure of contents related
  to :ref:`tools-Sphinx` and :ref:`tools-Jupyter`, and provides
  :ref:`syntax guidelines <concepts-documentation-style>`
* :ref:`concepts-code` describes the structure of contents related to
  :ref:`Python source code <tools-python>` and
  :ref:`pytest test code <tools-pytest>`, as well as some
  :ref:`syntax guidelines <concepts-code-style>`

.. tip::

   See :ref:`the links <references-links>` at any time for a centralized and
   comprehensive list of online resources relating to any of these topics
